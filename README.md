# Simple

A small educational repository of simple console programs implemented in multiple languages.

The goal of this repo is to show the same beginner-friendly problem solving ideas across a diversity of languages so learners can compare syntax, structure, and style.

## Languages Included

- Dart
- Java
- JavaScript
- Kotlin
- Python
- Swift

## Repository Structure

- `dart/`
- `java/`
- `javascript/`
- `kotlin/`
- `python/`
- `swift/`

## Program Topics

Most language folders include versions of these programs:

- Airplane ticketing / seat booking
- Basic calculations
- Marks average (pass/fail)
- Methods / function example
- Months
- Palindrome checker
- Student grades

Python currently also includes extra practice files:

- `area.py`
- `employee.py`
- `Hotel.py`

## Why This Repo Is Special

- Same learning ideas, many languages: easy side-by-side comparison.
- Beginner-focused logic: short, practical console examples.
- Useful for students, bootcamp practice, and language switching.

## How To Run

Run files from the repository root.

### Dart

```bash
dart run dart/<file>.dart
```

### Java

```bash
javac java/<FileName>.java
java -cp java <ClassName>
```

### JavaScript

```bash
node javascript/<FileName>.js
```

### Kotlin

```bash
kotlinc kotlin/<FileName>.kt -include-runtime -d app.jar
java -jar app.jar
```

### Python

```bash
python python/<file>.py
```

### Swift

```bash
swift swift/<FileName>.swift
```

## Suggested Learning Workflow

1. Pick one topic (for example, palindrome checker).
2. Read the same topic in 2-3 languages.
3. Compare input handling, conditionals, loops, and functions.
4. Modify each version and observe behavior differences.
